# Hello world python code
## Without a function 
Here is a python code to display "Hello world" without a function.
```python
    print("Hello world")
```

## With a function
Here is the python code to display "Hello world" with a function
```python
    def display_hello_world():
        print("Hello world")

    display_hello_world()
```
